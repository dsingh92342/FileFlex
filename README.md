**FileFlex** is a high-performance, local-first file transformer. Built with a stunning **Glassmorphism** aesthetic, it delivers a premium desktop-like experience entirely within your browser.

## üíé Why FileFlex?

Unlike traditional online converters, FileFlex processes your data **locally**. Your files are never uploaded to a server, ensuring 100% privacy, zero latency, and no file size limits.

## üöÄ Getting Started

No installation required. FileFlex is a static application.

1. **Clone the repo:** `git clone https://github.com/dsingh92342/FileFlex.git`
2. **Open `index.html`** in any modern evergreen browser (Chrome, Firefox, Safari, Edge).
3. **Drag & Drop** your files and start transforming.
